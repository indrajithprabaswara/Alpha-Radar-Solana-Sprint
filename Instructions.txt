Description
Most Pumpfun tokens on Solana are short-lived. Some get rugged, some fade with little traction, and a rare few take off.

This competition gives you access to 30 days of Pumpfun token transactions. The dataset has already been processed into a core set of transaction metadata and technical indicators, making it easier to build features while still staying close to the raw behavior of tokens.

In addition, we provide a CSV of target tokens ‚Äî those identified as profitable by a high-performing wallet‚Äôs strategy.

Your task:

Engineer features only from the provided Pumpfun dataset

Train a CatBoost model to predict with at least 75% accuracy, which tokens are likely to be part of the profitable subset

Ensure predictions are made within the first 30 seconds of a token‚Äôs lifespan (based on timestamp)

This challenge is:

Beginner-friendly: clean dataset with ready-to-use structure

Advanced: requires creative feature engineering and precision to minimize false positives

Practical: mirrors real-world token discovery pipelines in Solana

External data or alternative ML frameworks (LightGBM, XGBoost, deep learning, etc.) are not allowed.

Evaluation
Submissions are evaluated using the Jaccard Index (Intersection-over-Union), comparing your predicted positive set to the true target set on a private holdout. Provide a confusion matrix using Jaccard scoring in your results. Higher Jaccard indicates better performance.

Minimum 75% Recall on the private set is required for consideration.

Dataset Description
This dataset contains the first 30 seconds of post-mint transaction data for PumpFun tokens created in the month of September 2025.
The data was scraped using a proprietary parsing tool from a private Solana node.
The categories are fixed and must be adhered to.
